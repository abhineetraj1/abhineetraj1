## Cloud developer

Responsible for developing and maintaining applications that run on cloud computing platforms. This can include tasks such as designing, coding, testing, deploying, and updating cloud-based applications and services. Work with cloud platforms Amazon Web Services (AWS), and Microsoft Azure to create, configure, and manage cloud-based solutions. I use a variety of programming languages and tools - linux adminstration, Python (flask and django), javascript (vanilla, electron.js, node.js, mongoDB), Docker and Kubernetes, to build and deploy scalable and secure cloud applications.

Some of my typical responsibilities are:
*   Designing and building cloud-based applications and services
*   Migrating existing applications to the cloud
*   Managing and monitoring cloud infrastructure and services
*   Debugging and troubleshooting cloud-based applications
